# ![Study Snap Logo](https://github.com/metergames/study-snap/blob/c04707ffe730b824d5d0522e8aea90d93493ab71/StudySnap/Images/StudySnap%20Logo%2030.png) Study Snap

Study Snap is a study and quiz application that helps students review and retain concepts through flashcards, quizzes, and structured study sessions. It began as a school project and is now continued as an independent personal project.

## Features

- Create and manage flashcard decks
- Study mode for reviewing concepts
- Progress tracking and session history
- Search and filter decks
- Local data persistence

## Tech Stack

- Language: C#
- Framework: WPF
- Data: JSON-based local storage
- Architecture: Object-oriented design

## Project Status

This project is actively being improved. Planned work includes:
- AI-assisted flashcard creation
- UI and UX refinements
- Performance optimizations
- Improved analytics and insights
- Cloud sync and backup
